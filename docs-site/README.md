
# MtaaDAO Documentation Site

Interactive documentation built with Nextra (Next.js + MDX).

## Local Development

```bash
cd docs-site
npm install
npm run dev
```

Visit http://localhost:3001

## Build for Production

```bash
npm run build
npm start
```

## Structure

- `/pages` - MDX documentation files
- `/components` - React components (interactive demos, cards, etc.)
- `/styles` - Global CSS with MtaaDAO branding
- `theme.config.tsx` - Nextra theme configuration

## Features

- ✅ Interactive DAO type comparison
- ✅ Animated page transitions (Framer Motion)
- ✅ Search functionality
- ✅ Mobile responsive
- ✅ Code syntax highlighting
- ✅ Live component demos
- ✅ African cultural design elements

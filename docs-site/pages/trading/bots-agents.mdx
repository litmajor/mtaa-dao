# Trading Bot & Agent Development

Build autonomous trading agents for MtaaDAO.

## What is a Trading Agent?

A **trading agent** is software that:
1. **Monitors** market data (prices, volumes, indicators)
2. **Analyzes** using predefined rules or ML
3. **Executes** trades automatically
4. **Manages** risk (stops, position sizing)
5. **Learns** from results (optional)

## MtaaDAO Trading Agents

### Available Agents

1. **Mean Reversion Agent** - Trades range-bound markets
2. **Momentum Breakout Agent** - Trades trending markets  
3. **Volatility Regime Agent** - Adapts to market conditions
4. **Pairs Trading Agent** - Market-neutral strategies
5. **Anomaly Detection Agent** - Finds unusual patterns
6. **Liquidity Flow Tracker** - Exploits supply/demand
7. **Bayesian Belief Updater** - Probabilistic analysis

### Building Your Own Agent

```python
from mtaadao.trading import BaseStrategyAgent

class MyCustomAgent(BaseStrategyAgent):
    def __init__(self):
        super().__init__(name="MyStrategy")
        self.rsi_period = 14
        self.overbought = 70
        self.oversold = 30
    
    def update(self, market_data):
        """Analyze market and generate signal"""
        rsi = calculate_rsi(market_data, self.rsi_period)
        
        if rsi < self.oversold:
            return {"signal": "BUY", "confidence": 0.8}
        elif rsi > self.overbought:
            return {"signal": "SELL", "confidence": 0.8}
        else:
            return {"signal": "HOLD", "confidence": 0.5}
```

## Advanced Features

### Multi-Agent Consensus
Combine multiple agents for robustness:

```python
agents = [
    MeanReversionAgent(),
    MomentumBreakoutAgent(),
    VolatilityRegimeAgent()
]

signals = [agent.get_signal() for agent in agents]

# Execute only if 2+ agents agree
if sum(1 for s in signals if s["signal"] == "BUY") >= 2:
    execute_trade("BUY")
```

### Dynamic Position Sizing
Adjust position based on market conditions:

```python
def calculate_position_size(confidence, volatility, account):
    base_size = account * 0.02  # 2% base risk
    
    # Increase position if high confidence
    size = base_size * (0.5 + 1.5 * confidence)
    
    # Reduce position in high volatility
    size = size / (1 + volatility)
    
    return size
```

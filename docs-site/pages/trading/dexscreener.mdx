# DexScreener Integration

Discover and analyze tokens using DexScreener data.

## What is DexScreener?

**DexScreener** is a real-time DEX analytics platform that shows:
- Token prices across all DEXs
- Liquidity pools and trading volume
- Trending tokens and emerging opportunities
- Candlestick charts with technical indicators
- Community sentiment and metrics

## Why Use DexScreener?

| Feature | Benefit |
|---------|---------|
| **Real-time Data** | Update every few seconds |
| **Multi-DEX** | See prices across all chains |
| **Liquidity Metrics** | Know if pool is reliable |
| **Charts** | Technical analysis on any token |
| **Alerts** | Get notified of price moves |
| **Trending Tab** | Find emerging opportunities |

## MtaaDAO Integration

### API Connection

```python
from mtaadao.integrations import DexScreenerAPI

ds = DexScreenerAPI()

# Get token data
token = ds.get_token("celo", "0x123...")
print(f"Price: ${token['price']}")
print(f"Volume 24h: ${token['volume_24h']}")
print(f"Liquidity: ${token['liquidity']}")
```

### Dynamic Watchlist Update

```python
# Rebalance watchlist based on performance
for token in my_watchlist:
    current_data = ds.get_token(token)
    
    if current_data["volume_24h"] < 10_000:
        my_watchlist.remove(token)  # Remove low volume
    
    if current_data["price_change_24h"] > 100:
        my_watchlist.add_to_top(token)  # Feature emerging tokens
```

## Safety Tips

✓ Always verify contract address on multiple sources
✓ Check if contract is verified on block explorer
✓ Ensure liquidity is sufficient for your trade size
✓ Monitor for rug pull indicators (lp locked, team verified)
✓ Use slippage alerts to prevent sandwich attacks
✓ Start with small trades on new tokens

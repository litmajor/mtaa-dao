
# Cross-Chain Expansion: TRX, TON & Universal Swaps

MtaaDAO now supports cross-chain operations across multiple blockchains including TRON (TRX) and TON, with seamless token swapping capabilities.

## Supported Chains

### Mainnets
- **Celo** - Low-cost mobile-first blockchain
- **Ethereum** - Leading smart contract platform
- **Polygon** - High-speed, low-cost L2
- **BNB Chain** - Fast and efficient DeFi hub
- **Optimism** - Ethereum L2 with optimistic rollups
- **Arbitrum** - Scalable Ethereum L2
- **TRON (TRX)** - High-throughput blockchain (NEW)
- **TON** - Telegram Open Network (NEW)

### Testnets
- Celo Alfajores
- Polygon Mumbai
- BSC Testnet
- TRON Shasta
- TON Testnet

## Features

### 1. Cross-Chain Bridging

Transfer assets between any supported chains with minimal fees:

```typescript
// Bridge CELO to Ethereum
const transfer = await apiPost('/api/cross-chain/transfer', {
  sourceChain: 'celo',
  destinationChain: 'ethereum',
  tokenAddress: '0x...',
  amount: '100',
  destinationAddress: '0x...'
});
```

### 2. Cross-Chain Swaps

Swap tokens across different chains in a single transaction:

```typescript
// Swap CELO for TRX
const quote = await apiPost('/api/cross-chain/swap/quote', {
  fromChain: 'celo',
  toChain: 'tron',
  fromToken: 'CELO',
  toToken: 'TRX',
  fromAmount: '100',
  slippageTolerance: 1.0
});

// Execute the swap
const swap = await apiPost('/api/cross-chain/swap/execute', {
  quote,
  userAddress: '0x...'
});
```

## How It Works

### Bridge Protocol Stack

```
┌─────────────────────────────────────┐
│      User Initiates Transfer        │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│   Get Quote & Route Optimization    │
│   - Price feeds from oracles        │
│   - Best route calculation          │
│   - Fee estimation                  │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│      Lock Assets on Source Chain    │
│   - Smart contract escrow           │
│   - Emit bridge event               │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│    Bridge Relayers (LayerZero/      │
│    Axelar/Wormhole)                 │
│   - Cross-chain message passing     │
│   - Multi-sig validation            │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│   Mint/Release on Destination       │
│   - Verify proof                    │
│   - Execute swap if needed          │
│   - Transfer to recipient           │
└─────────────────────────────────────┘
```

### Security Features

1. **Multi-Signature Validation** - Transfers require multiple relayer confirmations
2. **Time-Locked Escrow** - Funds locked until destination confirmation
3. **Proof Verification** - Cryptographic proofs validate cross-chain messages
4. **Slippage Protection** - Maximum acceptable price movement
5. **Gas Estimation** - Accurate fee calculation before execution

## Swap Routing

The system automatically finds the best route for your swap:

```
CELO → TRX
├─ Direct Route (if pools exist)
│  └─ CELO → Bridge → TRX
│
└─ Optimal Route (via intermediary)
   └─ CELO → USDT → Bridge → TRX
```

## Usage Examples

### Example 1: Emergency TRON Transfer
```typescript
// Fast, cheap transfer for emergencies
const emergency = await apiPost('/api/cross-chain/transfer', {
  sourceChain: 'celo',
  destinationChain: 'tron',
  tokenAddress: 'USDT_CELO',
  amount: '100',
  destinationAddress: 'TAddress...'
});
// Completes in ~30 seconds
```

### Example 2: Arbitrage Opportunity
```typescript
// Exploit price differences between chains
const arbQuote = await apiPost('/api/cross-chain/swap/quote', {
  fromChain: 'polygon',
  toChain: 'bsc',
  fromToken: 'USDT',
  toToken: 'BUSD',
  fromAmount: '10000'
});

if (arbQuote.priceImpact < 0.5) {
  await apiPost('/api/cross-chain/swap/execute', {
    quote: arbQuote,
    userAddress: '0x...'
  });
}
```

### Example 3: Multi-Chain Vault
```typescript
// Create vault spanning multiple chains
const vault = await apiPost('/api/cross-chain/vault', {
  chains: ['celo', 'tron', 'ton'],
  name: 'Global Yield Vault'
});
```

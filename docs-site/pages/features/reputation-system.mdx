## Reputation Decay

### How Decay Works

**Inactivity Penalty**:
```typescript
const decayRules = {
  noActivity7Days: -0, // grace period
  noActivity30Days: -5, // per day
  noActivity90Days: -10, // per day
  maxDecay: 50 // percentage of total
};
```

**Example**:
```
Alice has 2,000 reputation
Inactive for 45 days:

Days 1-7: No decay (grace period)
Days 8-30: -5/day √ó 23 days = -115 rep
Days 31-45: -10/day √ó 15 days = -150 rep

Total decay: -265 rep
New score: 1,735 rep
```

**Preventing Decay**:
- Vote on at least 1 proposal per week
- Complete 1 task per month
- Comment on proposals
- Attend DAO meetings

## Reputation in Governance

### Weighted Voting

**Reputation Boost**:
```typescript
function calculateVotingPower(tokens: number, reputation: number) {
  const baseVotes = Math.sqrt(tokens);
  const repBonus = reputation / 100;
  return baseVotes + repBonus;
}

// Example
Alice: 10,000 tokens, 2,000 rep
Base: ‚àö10,000 = 100 votes
Bonus: 2,000 / 100 = 20 votes
Total: 120 votes (+20% from reputation)
```

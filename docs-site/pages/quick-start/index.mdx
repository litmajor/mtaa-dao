# Quick Start Guide

Get your DAO up and running in **less than 5 minutes**.

## What You'll Need

- Phone number or email
- DAO name (required)
- 2-3 co-founders (optional but recommended)
- Mobile money account (for treasury)

## 3-Step Setup

### Step 1: Create Your DAO (1 minute)
1. Open MtaaDAO app
2. Tap "Create DAO"
3. Enter: DAO name, description, location
4. Set initial treasury address

### Step 2: Invite Members (2 minutes)
1. Go to "Members"
2. Share invite link via WhatsApp, SMS, or Telegram
3. Add co-founders with admin roles
4. Set member roles and permissions

### Step 3: Make Your First Decision (1-2 minutes)
1. Go to "Proposals"
2. Create proposal: "Set monthly meeting day"
3. Members vote via SMS or app
4. Decision executed automatically

# Blockchain 101

Understand the technology powering MtaaDAO.

## What is Blockchain?

Blockchain is a **distributed ledger**—a record-keeping system spread across many computers, where:

1. **Transactions** are grouped into "blocks"
2. **Blocks** are cryptographically linked (hence "chain")
3. **Network** validates transactions together
4. **Immutability**: Once recorded, data can't be changed
5. **Transparency**: Everyone can see the entire history

## How It Works

### 1. Transaction Initiated
You create a transaction: "Send 100 MTAA to Alice"

### 2. Broadcasting
Transaction is broadcast to the network

### 3. Validation
Network nodes verify the transaction is valid:
- Do you have 100 MTAA?
- Is the signature correct?
- Are there no double-spends?

### 4. Consensus
Nodes agree on transaction validity through consensus (Proof of Work, Proof of Stake, etc.)

### 5. Block Creation
Valid transactions are grouped into a new block

### 6. Block Added
New block is added to the chain

### 7. Immutable Record
Transaction is now permanent and transparent

## Key Properties

### Decentralization
- No single authority
- Decisions made collectively
- Censorship-resistant

### Transparency
- All transactions public
- Everyone can verify
- Audit trail built-in

### Security
- Cryptographic hashing
- Immutable records
- No single point of failure

### Accessibility
- Open to anyone
- No permission needed
- Global participation

## Types of Blockchains

### Public (Permissionless)
- Anyone can participate
- Fully decentralized
- Examples: Bitcoin, Ethereum, Celo

### Private (Permissioned)
- Controlled access
- Higher throughput
- Examples: Enterprise blockchains

### Hybrid
- Public + private elements
- Custom controls

## Why Blockchain Matters for DAOs

✓ **Transparency**: Every transaction is visible
✓ **Trust**: No intermediary needed
✓ **Automation**: Smart contracts execute automatically
✓ **Accessibility**: Works globally 24/7
✓ **Cost**: No middlemen = lower fees

## Common Misconceptions

**"Blockchain = Bitcoin"**
No. Bitcoin uses blockchain, but blockchain has many applications.

**"Blockchain = Anonymous"**
No. It's pseudonymous—addresses are public, but linked to identities through exchanges.

**"Blockchain = Slow"**
No. Modern blockchains process thousands of transactions per second.

**"Blockchain = Unsafe"**
No. It's cryptographically secure. Hacks happen at exchange/wallet level, not blockchain level.

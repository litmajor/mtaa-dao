## Use Cases

### Treasury Prediction
```typescript
// User asks: "What will our treasury balance be in 30 days?"
const prediction = await nuru.predictTreasuryGrowth(daoId);
// Returns: predicted balance with confidence score
```

### Proposal Risk Assessment
```typescript
// User submits a high-value proposal
const risk = await nuru.assessRisk(proposalId, daoId);
// Returns: risk factors and recommendations
```

### Natural Language Queries
```typescript
// User asks: "Pesa zetu ni ngapi?" (Swahili for "How much money do we have?")
const response = await morio.handleMessage({
  userId,
  daoId,
  content: "Pesa zetu ni ngapi?"
});
// Returns: Treasury balance in user's language
```

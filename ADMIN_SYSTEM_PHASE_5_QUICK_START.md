# Phase 5: Agents & Elders Management - Quick Start Guide

## 🚀 Quick Navigation

### Access the Dashboard
1. Go to **Admin Panel** → `/admin`
2. Click **"Agents & Elders"** in the sidebar menu
3. You'll see the main dashboard with three tabs

### Three Main Views

#### 👑 **Elders Tab** (Default)
Shows your 3 specialized Elders with their stats:
- **KAIZEN** ⚙️ - Optimization expert
- **SCRY** 🔍 - Security & Risk expert  
- **LUMEN** ⚖️ - Ethics & Fairness expert

**What you see**:
- Elder cards with emoji icons
- Status (active/inactive)
- Uptime percentage
- Quick statistics (proposals analyzed, threats detected, etc.)

**To see details**:
- Click any elder card
- View extended statistics
- See recent actions/recommendations
- View complete activity history

#### 🤖 **Agents Tab**
View all operational agents:
- Analyzer (📊 Analysis)
- Defender (🛡️ Security)
- Scout (👀 Monitoring)
- Coordinator (🔄 Orchestration)
- Kwetu (💬 Community)

**What you see**:
- Agent status (online/offline)
- Messages processed count
- Average response time
- Error rate percentage
- Uptime tracking

**To see details**:
- Click any agent card
- View performance metrics
- See capabilities list
- Check recent activity logs

#### ⚙️ **Configuration Tab**
Manage system settings:
- Per-elder configuration
- Per-agent settings
- System-wide parameters

## 🎮 Main Controls

### Refresh Button
- Located in top-right corner
- Reloads all data from server
- Shows "Loading..." state while fetching

### Tab Navigation
- Click tabs to switch views
- Tabs: **Elders** | **Agents** | **Configuration**
- Current tab is highlighted in purple

### Detail Navigation
- Click any card to see full details
- Click **"← Back"** button to return to list view
- Use browser back button as alternative

## 💡 Common Tasks

### Check Elder Status
1. Go to **Elders** tab
2. Look for green **Active** badge
3. Check uptime % in top-right of card
4. Click for detailed performance history

### Monitor Agent Health
1. Go to **Agents** tab
2. Check status badge (green = online)
3. Review error rate % for issues
4. Click for activity logs and diagnosis

### View System Configuration
1. Click **Configuration** tab
2. Scroll through three sections:
   - **Elders Configuration**: Per-elder settings
   - **Agents Configuration**: Per-agent settings
   - **System Settings**: Global parameters

### Track Activity
1. Click any elder or agent card
2. Scroll to **Recent Activity** section
3. See timestamps and outcomes
4. Filter or search if needed

## 🆘 Troubleshooting

### Cards Not Loading?
- Click **Refresh** button
- Check network connection
- Verify admin permissions

### Status Shows "Offline"?
- This might be normal for maintenance
- Check elder/agent logs for details
- Contact system administrator if persistent

### Configuration Won't Update?
- Ensure you have admin privileges
- Try refreshing page
- Check browser console for errors

### Mobile Display Issues?
- Clear browser cache
- Try rotating device
- Update browser to latest version

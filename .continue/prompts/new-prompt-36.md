Audit indexing strategy.

Check:
- missing indexes
- redundant indexes
- incorrect composite indexes
- write-performance penalties
- query patterns not supported by indexes
- index selectivity issues
- index maintenance overhead
- index storage costs
- index usage patterns

Output:
- current weaknesses
- optimal index strategy
- tradeoffs
- expected performance improvements

---
name: Ship Readiness - Core
description: Validate core functionality
invokable: true
---

Determine if this feature is ready to ship.

Check:
- does the core idea work end-to-end?
- are critical paths functional?
- is user value delivered?
- are there any showstoppers or blockers?
- is the implementation stable enough for release?
- are there any critical bugs or issues that would prevent shipping?
  

Output:
- ready / not ready
- blocking issues
- next steps to address blockers
